# Blog-It

A dynamic blogging web application built using Node.js, Express.js, and EJS.

---

## Tech-Stack
- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript ES6+

---

## Features
- Create blog posts
- Edit blog posts
- Delete blog posts
- Dynamic rendering using EJS
- Express route handling
- Middleware usage
- Responsive design
- In-memory data storage
